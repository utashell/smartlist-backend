# 📝 SmartList (Backend)

A simple To-Do list app with AI-powered task suggestions.  
This repo contains the **frontend** built with React + Tailwind CSS.

---

## 🚀 Features
- Add / edit / delete tasks
- Basic task list UI
- Connects to backend API
- Beginner-friendly project

---

## 🛠️ Tech Stack
- React (Vite)
- Tailwind CSS
- Axios (for API calls)
