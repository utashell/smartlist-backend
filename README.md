# 📝 SmartList (Backend)

The backend API for the SmartList app.  
Built with **Node.js + Express + LowDB** for beginners.

---

## 🚀 Features
- REST API for managing tasks
- Simple JSON-based database (LowDB)
- Connects with frontend (React app)
- Beginner-friendly project

---

## 🛠️ Tech Stack
- Node.js
- Express
- LowDB
- CORS
